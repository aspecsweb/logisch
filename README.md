# Logisch

>Built for [www.logier.dev](https://www.logier.dev)

## Description

This project contains log parsers for various log formats.

## Local Dev Setup

Provide steps for setting up the project locally:

1. Clone the repository:
   ```sh
   git clone https://github.com/aspecsweb/logisch.git
   ```
2. Navigate to the project directory:
   ```sh
   cd logisch
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Develop locally ([Docs](https://docs.npmjs.com/cli/v9/commands/npm-link)):
   ```sh
   npm link
   ```
